# SmartDiary — Telegram бот для учета дедлайнов

## Описание проекта

SmartDiary — это Telegram-бот для студентов, который помогает отслеживать учебные дедлайны и важные события.

Бот позволяет удобно добавлять, просматривать и удалять дедлайны, а также быстро получать список ближайших задач.


## Проблема

Студенты часто забывают о дедлайнах, записывают их в разных местах или вовсе не фиксируют.

Это приводит к:
- пропущенным заданиям
- стрессу перед сдачей
- потере баллов


## Решение

SmartDiary решает эту проблему, предоставляя:
- простой интерфейс прямо в Telegram
- быстрый ввод дедлайнов
- просмотр актуальных задач
- автоматическую фильтрацию прошедших дедлайнов


## Функционал

Бот поддерживает следующие команды:

- `/start` — запуск бота
- `/add_deadline` — добавить новый дедлайн (пошаговый ввод)
- `/list` — показать все активные дедлайны
- `/nearest` — показать ближайшие дедлайны
- `/delete` — удалить запись по ID
- `/cancel` — отменить текущую операцию


## Особенности 

- Пошаговый ввод данных (conversation flow)
- Валидация даты и времени
- Время — опциональное поле
- Автоудаление прошедших дедлайнов
- Хранение данных в JSON-файле
- Обработка ошибок

## Промт 
Бот для учёта учебных дедлайнов, можно назвать : SmartDiary

Что делает:
1) сохраняет предмет или событие если не рушит логику и дедлайн (дата время если не рушит логику, хочу чтобы время было опционально но не обязательно)
2) показывает ближайшие дедлайны
3) удаляет запись (Автоудаление если технически возможно или не делать настоящее фоновое автоудаление, а сделать мягкое автоочищение: когда пользователь вызывает /list или /nearest бот сначала удаляет все просроченные записи из JSON потом показывает актуальные)
